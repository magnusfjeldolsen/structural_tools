Privacy: API keys you paste here stay in this browser tab. They are sent only to the provider's endpoint shown for each card. They are not logged to console, not persisted to localStorage, and not echoed in error messages. Closing this tab forgets them.
What each test does 
1. Text       — POST a "say hello" prompt; assert HTTP 200 + non-empty content.
2. Tool calls — POST with a toy tool definition (single number param); assert the model
                returns a tool call (name + parsed args).
3. Vision     — POST with a 1×1 red PNG embedded as base64; assert HTTP 200 + non-empty
                content. (Most Groq / Llama-class models reject this — that's expected.)
