# Plans Index

This directory holds feature plans produced by the `feature-planner` agent. Each plan is a self-contained, executable specification — a contract between intent and implementation. One plan per file, kebab-case slug, prefixed with the target area (e.g. `2dfea-…`, `concrete-beam-design-…`). Plans that have shipped are left in place as historical reference, not deleted.
