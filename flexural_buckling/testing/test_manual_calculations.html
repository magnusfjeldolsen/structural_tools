🎯 Objective

Verify that the flexural_buckling_api.js implementation matches manual calculations from SMath Studio for Class 4 effective properties.

Key parameter: ρ (rho) - reduction factor that determines how much of the plate remains effective

  • ρ = 1.0 → Fully effective (no removal)
  • ρ = 0.5 → 50% effective (half removed)
  • ρ = 0.0 → Completely ineffective (all removed)

🔍 Root Cause

The Problem: Code uses EN 1993-1-1 lambda for EN 1993-1-5 rho formula Current code (WRONG): const lambda_p_bar = (c/t) / (42 × ε); // EN 1993-1-1 const rho = (lambda_p_bar - 0.22) / lambda_p_bar²; // EN 1993-1-5 formula with wrong lambda! Should be: const lambda_p = (c/t) / (28.4 × ε × sqrt(k_σ)); // EN 1993-1-5 const rho = (lambda_p - 0.22) / lambda_p²; // EN 1993-1-5 formula with correct lambda
